Grid

Grid ใน CSS จะต้องกำหนดขึ้นมาเองโดยใช้ display:grid แล้วกำหนด column และ row ของ grid โดยใช้ grid-template-columns หรือ grid-template-row

BOOTSTRAP จะมี class ให้อยู่แล้วโดยใช้ class row เป้น parent element แล้ว chian เป็น col หรือ col-1 ถึง col-12

Flex

Flex ใน CSS จะต้องใช้ display:flex ก่อน หลังจากนั้นใช้ flex-direction: row หรือ column ใช้การกำหนดว่าจะให้เป็นแนวนอนหรือเเนวตั้ง

Flex ใน BOOTSTRAP จะมี class ให้อยู่แล้วโดยใช้ class d-flex จะเป็นการกำหนด display:flex และเราจะกำหนด row หรือ column โดยใช้ flex-row หรือ flex-column

margin & padding

CSS ถ้าเราจะกำหนด margin หรือ padding จะต้องกำหนดโดย margin:10px หรือ padding:10px จะโดย บน ขวา ล่าง ซ้าย หรือ ถ้าเราจะกำหนดซ้ายอย่างเดียวโดย margin-top | right | bottom | left:10px padding เช่นกัน

BOOTSTRAP จะมี class มาให้ โดย m คือ margin และ p คือ padding โดย BOOTSTRAP จะกำหนดมาให้ 5 เลข โดยจะใช้ m-1 ถึง 5 จะโดยทุกด้าน ถ้าอยากเฉพาะด้าน mt | r | b | l - 1 ถึง 5 padding เช่นกัน
