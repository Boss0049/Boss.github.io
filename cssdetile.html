CSS REVIEW

CSS syntax ประกอบด้วยอะไรบ้าง

CSS syntax ประกอบด้วย 3 อย่าง

  1. Selecter
  2. Prpperty
  3. Value

การเขียน CSS มีกี่แบบอะไรบ้าง

การเขียน CSS มี 3 แบบ

  1. Inline
  2. External
  3. Internal

selector มีกี่แบบอะไรบ้าง

selector มี 6 แบบ

  1. ID
  2. Class
  3. Pseudo element
  4. Pseudo class
  5. Combinators
  6. Attribute selector

combinator มีกี่แบบอะไรบ้าง

combinator มี

  1. space
  2. +
  3. ~
  4. >

property และ value คืออะไรแตกต่างกันอย่างไร

property และ value คือ

  1. property คือ ตัวคำสั่ง เช่น width,height
  2. value คือ ค่าคำหนดของ propety เช่น width:100% 100% คือ value

property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง

property มี 5 กลุ่ม

  1. property for content : color , size ,font ,text
  2. property for border : border ,table
  3. property for padding : padding
  4. property for margin : margin
  5. property for element : display ,Heigth ,Width ,Position ,Float ,Align

value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร

value unit มี 2 ประเภท

  1. Absolute lenghts : ค่าที่ฟิค เช่น px , cm , in , pt , pc
  2. Reletive lenghts : ค่าไม่ฟิค เช่น em , ex , vh , vw , %
